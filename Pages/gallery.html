P5.js Visual Artifact

Image, Sound and Code

24/11/2022

During my first year of university, I had the opportunity to explore the fascinating world of p5.js and create an interactive spiral project that captured the essence of dynamic visualizations. I was able to bring my creative vision to life by generating an ever-evolving spiral that responded to user input. By manipulating parameters such as speed, color, and size, the spiral took on various mesmerizing forms, enticing users to interact and uncover its hidden patterns.

Audio Artefact

Image, Sound and Code

26/01/2023

I created a composition using the software Mini-Tidal, which presented limitations such as a restricted number of synthesizer options and a lack of visual interface. To overcome these, I experimented with different parameters and effects to generate unique sounds, and organized my code meticulously to delineate sections and transitions

C++ Utility App

Codelab I

27/01/2023

Towards the end of my first semester, I embarked on an exciting project that involved creating a vending machine program using the powerful C++ language. I designed an intuitive user interface that replicated the functionalities of a real vending machine. The program managed tasks like displaying product options, receiving user input, processing transactions, and dispensing items accordingly

Bitsy Game

Digital Storytelling

03/02/2023

As part of the Digital Storytelling module, I created a short game using Bitsy. I encountered a few limitations while working with it. Despite these limitations, I learned to leverage the strengths of the platform to deliver an enjoyable and meaningful gaming experience
